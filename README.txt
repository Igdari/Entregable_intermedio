## Titulo ##
Administración de Gimansio

## Breve descripción del proyecto ##
Este proyecto permite realizar tareas simples de adminsitración de clientes, profesores y Planes
para un gimnasio deportivo
Con el se podra:
    Agregar nuevos clientes, profesores y planes de pago por semana para el gimnasio
    Listar los anteriores dichos para su mejor visualizacion

La tecnologia que se utilizo para realizar es el framwork de Django 4.1.5.
Modelado de HTML con librerias de bootstrap para el frontend 

## Como instalar y utilizar ##
Para comenzar a utilizar la herramienta siga los siguentes pasos
Programas requerido.
Editor de text (Preferiblemente Visual Studio)
Python
Django

Una vez instalado realizar un pull del gitHub https://github.com/Igdari/entregable_intermedio.git
Cuando se descarguen los archivos abra el editor de texto y abra la carpeta donde se encuentra
el proyecto

Ingrese por consola a la carpeta donde se encuentra el manage.Python
En consola escriba "python manage.py runserver" (Sin comillas)
Una vez inicie el servicio pordra acceder desde un explorador web al localhost:8000 
Y listo, ya tendra el acceso a la herramienta.

## Describiendo las secciones ##

Inicio 
Pagina inicial - Saludo al usuario

Agregar Cliente 
    Permite agregar clientes llenando los tres campos (Nombre, Apellido, Edad)
Listar Cliente  
    Trae todos los objetos que se encuentren en la base de datos de clientes. Los mismos pueden ser filtrados
Agregar Profesor   
    Permite agregar profesores llenando los tres campos (Nombre, Apellido, Si esta disponible o no para dar clases)
Listar Proferores
    Trae todos los objetos que se encuentren en la base de datos de profesores
Agregar Planes   
    Permite agregar planes de pago llenando los 2 campos (Nombre del plan, costo mensual)
Listar Planes
    Trae todos los objetos que se encuentren en la base de datos de Planes
